# SmugMug downloader

Recursively download all the images in a SmugMug gallery starting from a given folder nodeID.

# Usage

```
Usage of smugmug-downloader:
  -apiKey string
        APIKey
  -baseURL string
        base URL
  -nodeID string
        main nodeID
  -smsessCookie string
        SMSESS Cookie
```
